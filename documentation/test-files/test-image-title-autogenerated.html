Image Title Auto-Generated Detection Test

This page tests the IMAGE_TITLE_AUTOGENERATED rule which detects meaningless hover tooltips caused by CMSs auto-populating image title attributes.

FAIL Cases: A) Filename-Derived Title

Title matches the image filename after normalization (lowercase, remove extension, replace separators with spaces).

A1: Exact filename match (with separators converted to spaces)

title="kyle johnson tezzzuqtt8g unsplash"

Filename: kyle-johnson-TEZZzuQTt8g-unsplash.jpg

A2: Filename match with different casing

title="Kyle Johnson TEZZzuQTt8g Unsplash"

Should normalize to match filename

A3: Filename with underscores normalized

title="aditya saxena mixhvl wza unsplash"

Filename: aditya-saxena-_mIXHvl_wzA-unsplash.jpg

FAIL Cases: B) Title Contains File Extension

Title contains a file extension (.jpg, .png, .webp, etc.) indicating it was auto-generated.

B1: Title with .jpg extension

title="my-photo.jpg"

B2: Title with .png extension

title="screenshot-2024.png"

B3: Title with .webp extension

title="hero-banner.webp"

FAIL Cases: C) Camera/Screenshot Prefix

Title starts with common camera or screenshot prefixes (IMG_, DSC_, PXL_, Screenshot, etc.).

C1: IMG_ prefix

title="IMG_4920"

C2: DSC_ prefix

title="DSC_0042"

C3: PXL_ prefix (Google Pixel)

title="PXL_20250118_143022"

C4: Screenshot prefix

title="Screenshot 2025-01-18 at 3.22.10 PM"

C5: DCIM_ prefix

title="DCIM_100MEDIA"

FAIL Cases: D) Generic Placeholder Title

Title is a generic placeholder like "image", "untitled", "photo", or numbered variants.

D1: Generic "image"

title="image"

D2: Generic "untitled"

title="untitled"

D3: Generic "photo"

title="photo"

D4: Generic "picture"

title="picture"

D5: Numbered variant "image1"

title="image1"

D6: Numbered variant "photo-2"

title="photo-2"

D7: Numbered variant "img_3"

title="img_3"

PASS Cases: Valid Image Titles

These images should NOT trigger the rule.

P1: No title attribute

(no title attribute)

Images without title attributes are fine

P2: Empty title attribute

title=""

Empty titles are acceptable

P3: Meaningful descriptive title

title="A beautiful sunset over the mountain peaks"

Human-readable, meaningful title

P4: Product-specific title

title="2024 Annual Report Cover Design"

Specific, contextual title

P5: Title with "image" in context (not generic)

title="Hero image for the homepage redesign project"

Contains "image" but is clearly descriptive
